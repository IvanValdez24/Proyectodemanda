Bienvenido a mi Proyecto de Demanda de Energía

Explora cómo predecimos la demanda energética utilizando modelos avanzados como LSTM y datos climáticos.

Sobre el Proyecto

Este proyecto utiliza datos de demanda de energía y condiciones climáticas para entrenar un modelo LSTM que realiza predicciones precisas. Las variables clave incluyen temperatura, humedad, presión atmosférica, y velocidad del viento.

Ver en GitHub

El Dataset

  • Demanda de energía (SIN)
  • Temperatura
  • Humedad
  • Presión Atmosférica
  • Velocidad del viento a 10 metros

Los datos están divididos por horas y han sido escalados para optimizar el rendimiento del modelo.
